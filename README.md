# bookclub-backend


make sure to run

```
pip install requests
pip install flask
pip install flask-restful
pip install pymongo
```


first create a file named setup.xml

this contains the connection to your mongoDB server
and users for verification purposes

```xml

 <setup>
    <username></username>
    <password></password>
    <sever></server>
    <names>
        <name>Verfied User</name>
        <name>Verfied User</name>
    </names>
</setup>

```
